Franzboard - Raspberry Pi

This is the Raspberry Pi computer with an additional hardware extension shield.

We use it mainly for school projects. The github repositories contain some code examples for getting started.

Our extension shield named "franzboard" contains:

  • 4 LEDs
  • 3 Push buttons
  • 2 Onewire temperature sensors DS18B20
  • Remaining GPIO pins are accessible via the shield.
